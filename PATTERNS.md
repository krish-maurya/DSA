# C++ Pattern Programs – README

This repository contains **21 different pattern-printing programs** written in C++. Each pattern is implemented as a separate function (`pattern1` to `pattern21`).

All patterns depend on an integer input **n**.

For demonstration, the outputs below are shown assuming:

```
n = 4
```

1. Compile the code using a C++ compiler (g++)
2. Enter the value of `n` when prompted
3. Call the desired pattern function inside `main()`
